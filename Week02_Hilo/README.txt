---------------------------HiLo---------------------------------
------------------------Description---------------------------
 In the game Hilo, the player will be shown a card, and the player
gets to guess if the next card is going to be higher or lower! If the player's right 
they gain points, if not they lose points. See how many points you can get before you run out!

----------------------To Get Started------------------------------ 
Simply open the folder Week02_Hilo in VS Code, and then select Hilo.py 
and press "run"!

------------------------Structure-------------------------------
Director Class: Runs the game loop 
Card Class: Stores the value of the cards and their suit
Deck Class: Stores the cards and all the methods manipulating them
Hilo: the code to run to start the game

---------------------Required software-----------------------------
Python 3.8
